# Security Policy

## Reporting Vulnerabilities
If you discover a security vulnerability, please report it by emailing the maintainers.

## Supported Versions
| Version | Supported |
| ------- | --------- |
| 0.2.x   | Yes       |
| 0.1.x   | Yes       |

## Best Practices
- Never include API keys in prompts
- Validate all user inputs
- Use the PromptValidator to check for injection attacks
